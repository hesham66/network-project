# Enterprise Multi-Branch Network Design

This project showcases a full enterprise network infrastructure built with Cisco Packet Tracer. 
It connects multiple branches to a centralized HQ hosting DHCP, DNS, and Web servers.

## Features
- Centralized DHCP Server for all branches
- DNS Server resolving "hesham.local"
- Web Server hosting company intranet
- Static Routing with full end-to-end reachability
- IP Helper configuration for remote DHCP requests
